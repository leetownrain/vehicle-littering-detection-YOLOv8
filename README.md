# 🚗 車輛亂丟垃圾智慧偵測系統 
### Intelligent Vehicle Littering Detection System
## 🧩 系統簡介
本專案開發一套基於深度學習的「車輛亂丟垃圾智慧監測系統」，整合 YOLOv8 物件偵測、ByteTrack 多目標追蹤與垃圾拋擲事件判定機制，可即時辨識車輛違規拋棄垃圾行為。
系統具備：
- 即時車輛與垃圾偵測
- 違規事件追蹤與判定
- 靜態垃圾過濾機制
- 自動錄影存證
- LINE 即時違規通知
 <!-- -->
透過 OpenCV 環形緩衝錄影與 LINE Messaging API，系統可在偵測到違規時自動保存關鍵證據影片並即時推播通知。

## 🌟 核心技術與亮點
* **精準偵測與追蹤**：採用 YOLOv8 強化車輛與微小廢棄物之影像辨識，並透過 ByteTrack 分析物件之相對運動軌跡。
* **獨創邏輯判定**：
    * **時間窗口判定**：設定 2 秒窗口，若新出現的垃圾與移動車輛在相鄰區域，即判定具備關聯性。
    * **靜態垃圾過濾**：建立座標記憶機制，若新物件與既有垃圾座標相近則自動忽略，有效避免重複誤報。
* **全自動證據保存**：利用**環形緩衝錄影 (Ring Buffer)** 技術，捕捉違規發生前 2 秒與後續的關鍵影像，拼接成完整證據影片。
* **即時通報系統**：透過 **ngrok** 建立安全通道，並利用 **LINE Messaging API** 將證據影片連結即時推播至管理者手機。

## 🛠 技術棧
- **AI Model**: YOLOv8, ByteTrack
- **Language**: Python
- **API/Tools**: LINE Messaging API, ngrok, OpenCV (RTSP 分析)
